# Arrow Slither Escape PWA

Flat-file GitHub Pages PWA package.

Upload every file in this folder directly to your GitHub Pages site root. There is no separate icons folder in this version.

Files included:
- index.html
- manifest.webmanifest
- sw.js
- icon-192.png
- icon-512.png
- maskable-512.png
- apple-touch-icon.png
- icon-512.jpg

After upload, clear site data or hard refresh once if you previously tested an older build, because the browser may still have the old service worker/cache.
